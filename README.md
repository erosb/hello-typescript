Hello Typescript
================

Application skeleton for client-side apps.

It consists of the following components:

* typescript (language)
* gulp (build tool)
* bower (package manager)
* backbone (MVC framework)
* requirejs (module loader)
* handlebars (templating)
* jasmine (unit testing)

Prerequisites

Prerequisites
-------------

	npm install -g bower
	npm install -g bower-requirejs
	npm install -g gulp
	npm install -g tsd
	npm install -g typescript
	
Installation
------------

	git clone https://github.com/erosb/hello-typescript.git
	cd hello-typescript/
	npm install
	bower install
	tsd reinstall
	
Build commands
--------------

* gulp watch : sets up incremental compilation for development
